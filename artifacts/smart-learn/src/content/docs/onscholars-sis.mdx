# OnScholars SIS

The Student Information System (SIS) plugin for Moodle is designed to handle enrollments, grading periods, and student transcripts securely.

## Installation

1. Download the plugin zip file.
2. Extract it into your Moodle `/local` directory.
3. Visit `Site Administration > Notifications` to upgrade your database.

## Configuration

Navigate to `Local Plugins > OnScholars SIS` to configure the API keys and sync schedules.
