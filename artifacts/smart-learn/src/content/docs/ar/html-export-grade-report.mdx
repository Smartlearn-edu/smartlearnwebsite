تمكّن إضافة **تقرير تصدير الدرجات بصيغة HTML** (`gradereport_htmlexport`) المعلمين من تصدير درجات الطلاب بسهولة كملفات HTML متوافقة مع برنامج Word. من خلال الاحتفاظ بالعرض الهرمي للدرجات - المجمعة حسب الفئات والعناصر والمستخدمين - تعكس التقارير المصدرة بنية دفتر الدرجات المدمج في موودل. تحترم الإضافة بذكاء رؤية الدرجات، والعناصر المخفية، وإعدادات عرض الدرجات المخصصة.

## الميزات الرئيسية

- **تصدير فردي**: إنشاء تقارير درجات HTML مفصلة ومستقلة للطلاب الفرديين.
- **تصدير جماعي**: تنزيل درجات جميع الطلاب في وقت واحد في أرشيف ZIP مناسب.
- **هيكل هرمي**: يعكس بدقة مجموعات الدرجات والفئات والمجاميع.
- **تخصيص مرئي**: يوفر صفحة إعدادات مسؤول شاملة تحتوي على أكثر من **18 خيار ألوان مخصصًا** ومظاهر (بما في ذلك تنسيق التدرج الأرجواني الحديث).
- **العلامة التجارية للموقع**: يدمج شعار موقع موودل الخاص بك تلقائيًا مباشرة في التقارير.
- **إمكانية الوصول و RTL**: دعم كامل للغات من اليمين إلى اليسار (RTL) وتوافق مع مختلف المتصفحات.
- **تصميم جاهز للطباعة**: يتميز بـ CSS مخصص خصيصًا للطباعة النظيفة والتوافق مع Word.
- **التركيز على الخصوصية**: واجهة برمجة تطبيقات خصوصية مدمجة بسلاسة (متوافقة مع GDPR) لضمان التعامل الآمن مع البيانات.

## التثبيت

### التثبيت القياسي

1. **قم بتنزيل** ملف ZIP الخاص بالإضافة.
2. **استخرج** المحتويات إلى المسار التالي داخل تثبيت موودل الخاص بك:
   ```
   /path/to/moodle/grade/report/htmlexport/
   ```
3. **الترقية**: قم بزيارة **إدارة الموقع > الإشعارات** في موقع موودل الخاص بك لإكمال التثبيت وملء جداول قاعدة البيانات.
4. **التهيئة**: انتقل إلى **إدارة الموقع > الإضافات > تقارير الدرجات > تصدير HTML** لتخصيص سمات الألوان ودمج الشعار والإعدادات الأخرى.

## الصلاحيات والأذونات

يتم توفير الصلاحيات التالية بواسطة هذه الإضافة للتحكم في الوصول والرؤية:

- `gradereport/htmlexport:view`
  - **الوصف**: عرض واستخدام وظيفة تصدير HTML.
  - **الأدوار الافتراضية**: المعلمون، المعلمون المحررون، المديرون، الطلاب.

## الخصوصية

**هذه الإضافة لا تخزن أي بيانات شخصية.** 
إنها توفر بشكل صارم واجهة تصدير لبيانات الدرجات التي تديرها أنظمة موودل الأساسية بالفعل. يتم إنشاء ملفات HTML المصدرة ديناميكيًا ولا يتم تخزينها بشكل دائم على الخادم أبدًا.
