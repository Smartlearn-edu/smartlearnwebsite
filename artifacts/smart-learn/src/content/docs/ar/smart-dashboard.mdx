لوحة تحكم تحليلية قوية وشاملة لنظام موودل (Moodle).
مصممة **للمعلمين**، **والمديرين**، **والمسؤولين** — مع واجهة جميلة تدعم الوضع الداكن.

## لمحة عن الميزات

### نظرة عامة على المقررات
- بطاقات المقررات مع أعداد الطلاب وصور مخصصة
- متصفح الفئات للمسؤولين مع إحصائيات التسجيل
- لافتة ترحيب شخصية للطلاب مع شبكة أيقونات اختصارات (حتى 10 روابط قابلة للتخصيص)

### تقدم الطلاب والدرجات
- تتبع التقدم والإكمال عبر المقررات المتعددة
- تفاصيل إكمال الأنشطة لكل طالب بشكل معمق
- قائمة سجلات درجات الطلاب قابلة للتصفية
- تصدير مجموعات البيانات إلى ملفات CSV

### التصحيح وجدول الأعمال
- طوابير التصحيح المعلقة عبر جميع المقررات التي تدرسها
- تفصيل الواجبات/المهام مع تواريخ الاستحقاق
- **جدول أعمال اليوم (الخطة اليومية)**: مزامنة جدول مهام الطالب ديناميكيًا (يتكامل مع خطط الدراسة التكيفية)

### تحليلات النظام
- إحصائيات التسجيل على مستوى النظام بأكمله
- رسوم بيانية تحليلية على مستوى الفئات
- تصور لنسبة الطلاب إلى المعلمين
- تصدير السجلات إلى ملفات CSV

### تنبيهات الطلاب المعرضين للخطر
- تقييم المخاطر بشكل تركيبي عبر الإضافات الفرعية `smartdashboardrule`
- تتبع مقاييس تفاعل الطلاب (حداثة تسجيل الدخول، الإكمال، الدرجات، التأخير، الخطط التكيفية)
- التكامل مع خطافات الويب (Webhooks) في n8n للحصول على تنبيهات خارجية فورية

### التقارير السحرية (Magic Reports)
- معالج تقارير مخصص للمسؤولين
- إنشاء تقارير متقدمة تعتمد على SQL مع تصور للاستعلامات
- حفظ وتحميل وحذف الرؤى المخصصة

### تحليلات المدفوعات
- تتبع الإيرادات الفعلية مقابل التقديرية
- رسوم بيانية للإيرادات لكل فئة
- فلاتر للنطاق الزمني وتبديل للعملات
- تصدير السجلات إلى ملفات CSV

### الإعدادات والتهيئة
- تهيئة أيقونات الاختصارات المخصصة (الاسم، الفئة، الرابط)
- إعدادات وضع حساب المدفوعات
- إعدادات n8n Webhook (الرابط والرمز) لإشعارات المخاطر

### الأمان والخصوصية
- متوافق تمامًا مع **واجهة برمجة تطبيقات الخدمات الخارجية** في موودل (External Services API)
- ينفذ معايير **واجهة الخصوصية** الأساسية في موودل (Privacy API)
- التحقق الآمن من المعلمات وفحوصات الصلاحيات الصارمة

### تعدد اللغات
- متوفر جاهزًا باللغات **الإنجليزية**، **والإسبانية**، **والعربية**
- دمج كامل لوظيفة `get_string()` لتسهيل التعريب
- دعم تنسيق اللغات من اليمين إلى اليسار (RTL)

## التثبيت

### الخيار 1 — تنزيل ملف ZIP
1. قم بتنزيل أحدث ملف ZIP للإصدار.
2. في موودل، انتقل إلى **إدارة الموقع ← الإضافات ← تثبيت الإضافات**
3. قم برفع ملف ZIP واتبع التعليمات.

### الخيار 2 — الاستنساخ عبر Git
إذا كان لديك وصول إلى وحدة التحكم (Terminal) في الخادم، انتقل إلى مسار تثبيت موودل الخاص بك:

```bash
cd /path/to/moodle/local
git clone https://github.com/Smartlearn-edu/moodle_local_smartdashboard.git smartdashboard
```
ثم قم بزيارة **إدارة الموقع ← الإشعارات** لإكمال التثبيت.

> **هام:** يجب أن يكون اسم المجلد داخل مسار `/local/` هو **`smartdashboard`**.

## الاستخدام

بعد التثبيت، يمكنك الانتقال إلى لوحة التحكم الذكية عبر الرابط:

```
https://your-site.com/local/smartdashboard/
```

### الوصول حسب الدور

| الدور | ما تراه |
|---|---|
| **الطالب** | لافتة الترحيب، شبكة الاختصارات القابلة للتخصيص، والخطة اليومية/جدول الأعمال |
| **المعلم** | نظرة عامة على المقرر، تقدم الطلاب والدرجات، طابور التصحيح، تحليل المخاطر، والخطة اليومية |
| **المدير** | كل ما سبق + تحليلات الفئات، إحصائيات النظام |
| **المسؤول (Admin)** | وصول كامل يشمل تحليلات المدفوعات، تهيئة الإعدادات، والتقارير السحرية |

### إضافة كتل موودل (Blocks) إلى لوحة التحكم

افتراضيًا، تقيد بعض كتل موودل نفسها للظهور فقط في صفحات المقررات أو لوحة تحكم موودل القياسية. لجعل أي كتلة متاحة للإضافة إلى لوحة التحكم الذكية، تحتاج إلى إضافة نوع صفحة لوحة التحكم (`local-smartdashboard-*`) إلى التنسيقات المسموح بها للكتلة.

**مثال: تعديل كود الكتلة**
1. افتح ملف PHP الرئيسي للكتلة: `blocks/<blockname>/block_<blockname>.php`
2. ابحث عن الدالة `applicable_formats()`.
3. أضف `'local-smartdashboard-*' => true,` إلى المصفوفة المُرجعة.

```php
public function applicable_formats() {
    return [
        'course-view'            => true,
        'site'                   => true,
        'my'                     => true,
        'local-smartdashboard-*' => true, // <-- أضف هذا السطر
    ];
}
```
> **ملاحظة:** تأكد من **مسح جميع ذاكرات التخزين المؤقت** (إدارة الموقع ← التطوير ← مسح ذاكرات التخزين المؤقت) بعد حفظ الملف لترى الكتلة في قائمة "إضافة كتلة"!

## مرجع واجهة برمجة التطبيقات (API)

جميع الوظائف متاحة بشكل آمن عبر **واجهة برمجة تطبيقات الخدمات الخارجية** في موودل (`local_smartdashboard_webservice`):

- `local_smartdashboard_get_cross_course_progress` - بيانات تقدم الطلاب عبر المقررات
- `local_smartdashboard_get_student_detailed_progress` - تفاصيل إكمال الأنشطة لطالب واحد
- `local_smartdashboard_get_grading_overview` - المهام التي تحتاج إلى تصحيح
- `local_smartdashboard_get_system_analytics` - إحصائيات التسجيل على مستوى النظام للمسؤولين/المديرين
- `local_smartdashboard_get_payment_analytics` - تحليلات الإيرادات والمدفوعات
- `local_smartdashboard_save_dashboard_settings` - حفظ تفضيلات إعدادات لوحة التحكم
- `local_smartdashboard_get_dashboard_settings` - استرداد تهيئات لوحة التحكم الحالية
- `local_smartdashboard_get_cross_course_grades` - بيانات درجات الطلاب عبر المقررات
- `local_smartdashboard_get_magic_insight` - الحصول على رؤى التقارير المخصصة والاستعلامات المدعومة بالذكاء الاصطناعي
- `local_smartdashboard_save_magic_report` - حفظ تهيئة تقرير سحري مخصص
- `local_smartdashboard_get_saved_reports` - قائمة التقارير السحرية المحفوظة
- `local_smartdashboard_delete_magic_report` - حذف تهيئة تقرير سحري محفوظ
- `local_smartdashboard_get_programs` - قائمة البرامج ومعرفات المقررات المرتبطة بها
- `local_smartdashboard_dismiss_announcement` - إخفاء إعلان أخباري من لوحة التحكم
- `local_smartdashboard_get_daily_plan` - خطة الدراسة اليومية للطالب (تكامل الخطة التكيفية)
- `local_smartdashboard_get_risk_data` - بيانات تتبع تفاعل الطلاب المعرضين للخطر
